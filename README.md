This project presents a Machine Learning-based Network Intrusion Detection System (NIDS) developed using the CIC-IDS2017 dataset,The objective is to classify network traffic into Benign and Malicious categories by analyzing flow-based behavioral features.
The project implements an end-to-end ML pipeline including data integration, preprocessing, feature selection, model training, and performance evaluation.

Objectives
> Build an automated intrusion detection system using machine learning
> Reduce false negatives by prioritizing recall
> Improve model generalization through feature selection
> Compare multiple classification algorithms for performance analysis
